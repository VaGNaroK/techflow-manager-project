# TechFlow Solutions - LogiTask

Gerenciador de tarefas web leve para o fluxo de redespacho e entrega de mercadorias.

## Mudança de Escopo: Atributo de Prioridade
A pedido do cliente (LogiExpress), foi introduzido o campo de `priority` (Baixa, Média, Alta) no modelo de tarefas. 
**Justificativa de negócio:** A operação descentralizada precisava não apenas acompanhar o status (fluxo), mas também identificar o nível de urgência de cada tarefa para que a gerência pudesse focar nos gargalos operacionais mais críticos em tempo real.
